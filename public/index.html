2. 输入文本与生成

让克隆声音朗读你的文本

Edge TTS 免费音色：使用你部署的 Edge TTS Worker，支持大量微软原生中文、韩文、英文音色，不需要 Azure Speech Key。
ElevenLabs 免费 TTS：不上传音频、不克隆声音，直接使用 ElevenLabs 账号可用的官方音色做文字转语音。
ElevenLabs：支持上传几秒音频做声音克隆，需要在 Cloudflare 环境变量配置 ELEVENLABS_API_KEY。
Microsoft Azure：推荐用于原生普通话和原生韩语朗读，不使用上传样本克隆声音。可在 Cloudflare 配置 AZURE_SPEECH_KEY / AZURE_SPEECH_REGION。
Google Cloud：可选原生中文/韩文 TTS 音色，不使用上传样本克隆声音。建议在 Cloudflare 配置 GOOGLE_SERVICE_ACCOUNT_JSON。
1forall.ai：使用 speech/text-to-speech 接口生成普通朗读音频，鉴权格式为 Authorization: Api-Key。上传样本不会用于克隆。
自定义 API：会以 multipart/form-data 发送 text、voiceName、sample 字段，并期待接口直接返回音频文件。
